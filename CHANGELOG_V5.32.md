# SuperApp Famille Mobile — V5.32

## Démarrage propre

- Suppression de toutes les données personnelles ou de démonstration préremplies.
- Nouvelle installation vide : aucun membre, aucun foyer personnel, aucune tâche, aucune course, aucun stock, aucun repas, aucune donnée Santé, Éducation, Sport, Documents ou Notifications.
- Les textes d’exemple restent uniquement des aides dans les champs ; ils ne créent aucune donnée réelle.

## Réinitialisation complète

- Le bouton « Supprimer les exemples » et le bouton « Réinitialiser » ouvrent désormais une fenêtre de confirmation au design de l’application.
- La réinitialisation vide totalement l’application, y compris les membres du foyer et les paramètres familiaux.
- Avant suppression, l’utilisateur peut exporter ses données au format JSON.
- La fenêtre explique comment réimporter le JSON après réinitialisation : Paramètres > Sauvegarde et données > Importer JSON.

## Bienvenue / parcours guidé

- Première page de bienvenue rendue plus attractive avec une grande image et des icônes d’apps.
- Le message précise que l’application démarre proprement, sans données enregistrées.
- Pendant le guide, l’ajout d’un membre reste dans le flux de bienvenue : fermer ou enregistrer la fenêtre de membre ramène à l’étape du guide, pas aux Paramètres.

## Migration de sécurité

- Si une ancienne version contient encore les données personnelles de démonstration historiques, elles sont purgées automatiquement au premier lancement de cette version.
