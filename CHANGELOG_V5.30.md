# SuperApp Famille Mobile — V5.30

## Corrections ciblées

### Courses > Stocks
- Le bouton **Aux courses** n’ajoute plus directement le produit dans la liste.
- Une fenêtre de validation au design de l’application s’ouvre avant l’ajout.
- La fenêtre affiche le produit, le stock actuel, l’unité et la quantité proposée.
- Si le produit est déjà présent dans la liste de courses, aucun doublon n’est créé et l’utilisateur est informé.

### Paramètres > Notifications
- Les préférences de notifications sont maintenant appliquées dans la génération réelle des notifications.
- Si les notifications globales sont désactivées, aucune notification applicative n’est affichée.
- Si une application est désactivée dans les préférences de notifications, ses rappels n’apparaissent plus dans le centre de notifications ni sur l’accueil.
- L’état reste sauvegardé dans les données de l’application.

### Textes d’exemple
- Le champ principal “Quoi ?” affiche désormais un exemple adapté au module ouvert : Courses, Santé, Éducation, Sport, Maison, Famille ou Calendrier.
