# SuperApp Famille Mobile — V5.36.4

## Checklist — harmonisation transversale

- Suppression de l'ancien champ texte multiligne dans l'ajout Sport / Loisir / Voyage.
- Création et modification d'une activité utilisent maintenant le même parcours : enregistrer puis ouvrir la page checklist structurée.
- Le bouton « Enregistrer et ouvrir la checklist » lance directement la checklist liée à l'activité créée.
- Les checklists restent liées à leur activité : objet, quantité, catégorie, suggestions, statut coché/barré, suppression manuelle.
- Les listes cochables sont harmonisées : coché = visible + barré + statut clair, sans disparition automatique.
- Les vues Courses, Éducation et Tâches affichent aussi les éléments faits pour conserver la trace visuelle barrée.
